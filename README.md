# 🛒 Full Stack Online Order System (SQL Backend)

## 📌 Overview

This project is a **Full Stack Online Order System** developed using frontend technologies and a **SQL database backend**. It allows users to browse items, place orders, and store order data efficiently in a structured database.

---

## 🚀 Features

* 🧾 Browse available products/items
* 🛍️ Add items to cart / order
* 📦 Place and manage orders
* 🗄️ Data stored in SQL database
* 🔄 Dynamic interaction between frontend and backend
* 💻 Clean and responsive user interface

## ▶️ How to Run Locally

### 1. Clone Repository

```bash
git clone https://github.com/your-username/your-repo-name.git
```

### 2. Setup Database

* Open MySQL / phpMyAdmin
* Create a new database
* Import `database.sql` file

### 3. Run Backend Server

* If using XAMPP:

  * Start Apache & MySQL
  * Place project in `htdocs`

* If using Node.js:

  ```bash
  npm install
  npm start
  ```

### 4. Open in Browser

* Visit: `http://localhost/your-project-folder`
